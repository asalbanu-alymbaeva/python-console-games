# 🎮 Python Mini Games

A beginner-friendly collection of simple **Python CLI games**.  
This project includes interactive games that demonstrate basic programming concepts such as loops, conditionals, and user input handling.

---

## 🚀 Features

- 🎯 Guess the Number (User guesses computer's number)
- 🤖 Computer Guesses Your Number (AI-style guessing using binary search)
- 🔤 Guess the Word (Uzbek word guessing game)
- 📚 Uses a dataset of 30,000+ Uzbek words

---

## 📁 Project Structure
project-folder
│
├── script.py # Main file to run the games
├── functions.py # All game logic functions
